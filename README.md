a website to track your book reading


#### install python

```sh
brew install python
```

### check python version

```sh
python3 --version
```

### pip version

```sh
pip3 -v
```

### install libraries

```sh
pip3 install -r requirements.txt
```

### check code coverage
```shell
coverage run -m pytest
coverage html
```
